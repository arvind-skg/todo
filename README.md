📝 To-Do List Web App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.
This app helps users manage daily tasks with features like adding, deleting, marking as completed, and saving tasks using browser storage.

🚀 Features

➕ Add new tasks

❌ Delete tasks

✅ Mark tasks as completed

⏱️ Timestamp for each task

💾 Saves tasks using Local Storage (data persists after refresh)

📱 Fully responsive design

⌨️ Press Enter to quickly add tasks

🛠️ Technologies Used

HTML5

CSS3 (Responsive Design + Glassmorphism UI)

JavaScript (DOM Manipulation & Local Storage)

⚙️ How It Works

Tasks are created dynamically using JavaScript (createElement)

Data is stored in localStorage as JSON

On page load, saved tasks are retrieved and displayed

Each task includes:

Task text

Timestamp

Completion status

💡 Key Concepts Used

DOM Manipulation (appendChild, querySelector)

Event Handling (onclick, keydown)

Browser Storage (localStorage)

JSON Parsing & Stringifying

📸 UI Highlights

Glassmorphism effect for task cards

Custom background with overlay PNG

Mobile-friendly layout using media queries

▶️ How to Run

Download or clone the project

Open index.html in any browser

Start adding your tasks ✅
